# Tachymaze Website Development Agents

This file defines specialized AI agents to help modernize and enhance the Tachymaze marketing website. These agents are designed to work on the `tachymaze/` folder, transforming the current Bootstrap-based site into a modern, cool, and engaging experience.

## Web Design Modernization Agent

**Description:** Specializes in updating the visual design of the Tachymaze website to contemporary standards. Focuses on modern color schemes, typography, layouts, and user experience improvements.

**Use when:** Redesigning layouts, updating color palettes, improving typography, adding modern UI elements like cards, gradients, shadows, and micro-interactions.

**Capabilities:**
- Analyze current design and suggest modern improvements
- Implement CSS Grid and Flexbox for better layouts
- Add subtle animations and transitions
- Ensure mobile-first responsive design
- Optimize for accessibility and performance

## CSS Enhancement Agent

**Description:** Expert in modern CSS techniques to replace outdated styles with cutting-edge approaches. Transforms the current custom CSS into a more maintainable and feature-rich stylesheet.

**Use when:** Updating stylesheets, adding custom properties (CSS variables), implementing modern selectors, optimizing for performance.

**Capabilities:**
- Convert to modern CSS features (Grid, Flexbox, custom properties)
- Add dark mode support if appropriate
- Implement smooth animations and transitions
- Optimize CSS for faster loading
- Ensure cross-browser compatibility

## Interactivity and JavaScript Agent

**Description:** Adds dynamic features and interactivity to make the website more engaging. Implements vanilla JavaScript or modern libraries for enhanced user experience.

**Use when:** Adding interactive elements, form handling, animations, or dynamic content loading.

**Capabilities:**
- Add scroll-triggered animations
- Implement interactive features like image galleries or testimonials
- Create smooth page transitions
- Add form validation and submission handling
- Optimize JavaScript for performance

## Content Optimization Agent

**Description:** Improves the website's content structure, SEO, and messaging to better promote the Tachymaze app.

**Use when:** Updating text, improving meta tags, enhancing SEO, restructuring content for better flow.

**Capabilities:**
- Optimize headings and content hierarchy
- Improve meta descriptions and Open Graph tags
- Enhance call-to-action buttons and links
- Add structured data for better search visibility
- Ensure content is mobile-friendly

## Performance and Optimization Agent

**Description:** Focuses on making the website fast, lightweight, and optimized for all devices and network conditions.

**Use when:** Optimizing images, minifying code, improving loading times, ensuring accessibility.

**Capabilities:**
- Optimize images and assets
- Minify CSS and JavaScript
- Implement lazy loading
- Add accessibility features
- Test and improve Core Web Vitals

## Framework Migration Agent (Optional)

**Description:** If needed, migrates the static site to a modern framework like React, Vue, or Svelte for better maintainability and features.

**Use when:** The static approach becomes limiting and a dynamic framework would add significant value.

**Capabilities:**
- Assess if migration is beneficial
- Set up build tools and frameworks
- Convert existing HTML/CSS to component-based architecture
- Add state management if needed
- Ensure SEO and performance are maintained

## General Guidelines for All Agents

- Always maintain the focus on promoting the Tachymaze iOS app
- Keep the website lightweight and fast-loading
- Ensure full responsiveness across all devices
- Use semantic HTML and follow web standards
- Test changes thoroughly before deployment
- Preserve the app's branding and messaging
- Prioritize user experience and conversion optimization
